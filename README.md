# Deepak Dakshina Moorthy — HR & Employee Relations Portfolio

This repository hosts a clean HR and Employee Relations portfolio designed to support applications for Employee Relations Specialist, HR Advisor, People Advisor and People Operations roles.

## Positioning

**HR & Employee Relations Portfolio**  
UK ER Casework | Manager Advisory | HR Compliance | Process Improvement

## Portfolio focus

The site is structured around evidence that is more credible for HR roles than a generic CV website:

- Employee relations casework and documentation
- Manager advice and coaching
- Absence, welfare, probation and performance support
- Conduct, disciplinary, grievance and investigation support
- Right to work, DBS, employee data and staff file compliance
- CQC and local authority audit readiness
- HR process improvement using trackers, SharePoint, Google Forms and Power Automate

## Confidentiality standard

All examples uploaded or linked from this portfolio must be anonymised. Do not include names, addresses, dates of birth, employee numbers, payroll information, immigration details, service user details, resident details, relatives' details or live case identifiers.
